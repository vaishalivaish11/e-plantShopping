# 🌿 Paradise Nursery

Paradise Nursery is a React + Redux based plant shopping application.

## 🚀 Features
- Product listing with categories
- Add to cart functionality
- Redux Toolkit state management
- Cart quantity update system
- Dynamic cart counter
- Clean UI with navigation

## 🛠️ Tech Stack
- React JS
- Redux Toolkit
- React Router
- CSS

## ▶️ Setup Instructions
npm install
npm start
